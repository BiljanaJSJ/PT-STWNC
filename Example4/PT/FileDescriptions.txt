Brief descriptions of the files included in this directory:

gas-furnace.csv                 - Gas-furnace data
BNN_PT.R                        - main functions used to run the algorithm by runPT.R
BNN_PT_fun.R                    - help functions used to run the algorithm by runPT.R           
runPT.R                         - run the algorithm
PT_ThermodynInt_150000.RData    - samples obtained after running the algorithm 
Predictions1.RData              - predictions and MSPE estimates
checkTheResultsPT.R             - obtain predictions and MSPE estimates
