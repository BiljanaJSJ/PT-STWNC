Brief descriptions of the files included in this directory:

gas-furnace.csv                 - Gas-furnace data
BNN_ST.R                        - main functions used to run the algorithm by runST.R
BNN_ST_functions.R              - help functions used to run the algorithm by runST.R           
runST.R                         - run the algorithm
ST455000.RData                  - samples obtained after running the algorithm 
Predictions1_455.RData          - predictions and MSPE estimates
checkTheResults.R               - obtain predictions and MSPE estimates
CalcEES.R                       - calculate Effective Sample size adjusted for the 
                                  autocorrelation for both algorithms PT-STWNC and PT
