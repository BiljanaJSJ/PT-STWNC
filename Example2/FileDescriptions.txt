Brief descriptions of the files included in this directory:

runST.R                       - run the algorithm
ST_Galaxy_runST.R             - main functions used to run the algorithm by runST.R
ST_Galaxy_functions.R         - help functions used to run the algorithm by runST.R           
plotsGalaxyDataPaper.R        - make plots in the paper (Example 3) and save them into .eps format 
ST50000.RData                 - samples obtained after running the algorithm 




Subdirectories contain files of code used to run the marginal likelihood estimation 

PT_TI_marginalllik           - calculate marginal likelihood using Thermodynamic Integration with Parallel Tempering                                    
PT_STWTDNCmargllik           - calculate marginal likelihood using Thermodynamic Integration with STWNC
