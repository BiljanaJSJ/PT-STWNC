Brief descriptions of the files included in this directory:

runST.R                      - run the algorithm
ST_functions_new.R           - main functions used to run the algorithm by runST.R
PlotsParamEstimPaper.R       - make plots in the paper (Example 2) and save them into .eps format 
filledContourFunction1.R     - graphical function



Subdirectories contain files of code used to run the marginal likelihood estimation 

PT_TI_mllik                  - calculate marginal likelihood using Thermodynamic Integration with Parallel Tempering                                    
PT-STWNC_mllik               - calculate marginal likelihood using Thermodynamic Integration with STWNC                                    
WL_pants                     - run the Parallel adaptive Wang Landau algorithm using the same data and model
